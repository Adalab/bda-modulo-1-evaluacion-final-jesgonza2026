Tienda Online

Este proyecto implementa una clase llamada TiendaOnline que simula el gestionamiento de una tienda: permite agregar productos, buscarlos, actualizar stock, eliminarlos y calcular el valor total del inventario.



Funcionalidades principales:
    -Agregar productos
    -Ver inventario
    -Buscar productos
    -Eliminar productos_
    -Calcular valor total del inventario
